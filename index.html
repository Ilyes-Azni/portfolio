Model inversion attack through GANs

GAN-Based Model Inversion Framework that extracts class information from trained neural networks using generative adversarial networks. Uses latent space optimization instead of pixel-space approaches to generate realistic images that maximize target model confidence. Demonstrates privacy vulnerabilities across various model architectures including image classifiers and deepfake detectors. This work was presented at 2025 AmuSec at CIRM in Marseille.
